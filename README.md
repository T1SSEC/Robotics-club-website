# robotics club website
Website for our robotics club that shows our projects and fun histories.

## Project Structure

This project follows a simple web structure to organize our robotics club website. Here's the recommended file structure:

- `index.html` - The main homepage of the website
- `about.html` - Information about our robotics club
- `projects.html` - Showcase of our robotics projects
- `events.html` - Upcoming and past events
- `contact.html` - Contact information and how to join

- `css/` - Folder for stylesheets
  - `styles.css` - Main stylesheet for the website
  - `responsive.css` - Styles for mobile and tablet responsiveness

- `js/` - Folder for JavaScript files
  - `script.js` - Main JavaScript file for interactive features
  - `animations.js` - Scripts for animations and effects

- `images/` - Folder for all images and media
  - `logo.png` - Club logo
  - `robots/` - Subfolder for robot photos
  - `events/` - Subfolder for event photos
  - `projects/` - Subfolder for project images

- `assets/` - Folder for other assets
  - `fonts/` - Custom fonts if needed
  - `videos/` - Video files of our robots in action

- `README.md` - This file with project information
